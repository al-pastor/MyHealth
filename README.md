# My Health

Health Tracking Application

## Getting Started

* Npm install
* Npm start - Starts the Node Server
* Npm run webpack - Watches the webpack bundle file.

### Prerequisites

Must have [Node](https://nodejs.org/en/download/) installed to work with node and npm.
